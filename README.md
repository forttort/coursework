# Coursework

Черновой репозиторий проекта веб-каталога отобранных товаров с японских сайтов.

## Текущая идея
Проект представляет собой веб-приложение, в котором товары с японских интернет-площадок будут собираться, фильтроваться и публиковаться в виде каталога.

Каталог ориентирован на отобранные позиции, без массового вывода всего ассортимента источников. В дальнейшем пользователь сможет выбрать товар и оформить заказ через владельца сервиса.

## Предполагаемый стек
- Python
- PostgreSQL
- FastAPI
- Web client

## Статус
Проект находится на этапе формирования общей концепции, структуры и требований.
