# Coursework

Веб-каталог fashion-товаров с японских площадок.

## Описание проекта
Проект представляет собой веб-приложение для отображения товаров fashion-сегмента в формате каталога. Пользователь должен видеть товары в виде карточек и иметь возможность открыть подробную страницу каждого товара.

## Цель проекта
Создать прототип веб-каталога товаров fashion-сегмента с японских площадок, в котором товары отображаются с основной информацией, ценой в рублях и изображениями.

## Категории товаров
- одежда
- обувь
- сумки
- аксессуары
- украшения
- часы

## Основные функции
- отображение каталога товаров
- отображение карточек товаров
- поиск по товарам
- фильтрация по категориям
- просмотр подробной страницы товара

## Источники данных
Основные источники данных:
- Mercari Japan
- Yahoo! Auctions Japan
- Rakuten Rakuma
- 2nd STREET
- RINKAN

## Структура проекта
- `docs` — документация проекта
- `sql` — схема базы данных и SQL-скрипты
- `parser` — парсинг данных с внешних площадок
- `backend` — серверная часть и API
- `frontend` — пользовательский интерфейс
