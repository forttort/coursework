# PROJECT CONTEXT

## Название проекта
Веб-каталог fashion-товаров с японских площадок

## Общая идея
Проект представляет собой веб-приложение для отображения товаров fashion-сегмента в формате каталога.

## Цель проекта
Создать веб-каталог товаров fashion-сегмента с японских площадок, в котором товары отображаются в виде карточек с основной информацией.

## Категории товаров
- одежда
- обувь
- сумки
- аксессуары
- украшения
- часы

## Формат отображения
Система должна включать:
- каталог товаров
- карточки товаров
- поиск
- фильтрацию по категориям
- страницу товара с подробной информацией

## Данные товара
На карточке товара отображаются:
- фото
- бренд
- название
- категория
- размер
- состояние
- цена в рублях
- срок доставки

## Источники данных
Основные источники данных:
- Mercari Japan
- Yahoo! Auctions Japan
- Rakuten Rakuma
- 2nd STREET
- RINKAN

## Структура проекта
- `docs` — документация
- `sql` — SQL-скрипты
- `parser` — парсер данных
- `backend` — серверная часть
- `frontend` — интерфейс
