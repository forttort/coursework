# DECISIONS

## Зафиксированные решения
- проект связан с каталогом товаров с японских сайтов
- клиентская часть будет веб-приложением
- база данных — PostgreSQL
- backend — FastAPI
- репозиторий пока создается в черновом виде без жесткой фиксации всех требований

## Временные допущения
- детали проекта могут меняться
- структура базы данных пока не финальная
- состав модулей и функций будет уточняться позже
