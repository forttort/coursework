# FRONTEND

Описание текущего frontend MVP.

## Стек
- `HTML`
- `CSS`
- `JavaScript`

Frontend сделан без сборщика и без framework.

## Файлы
- `frontend/index.html` — каталог товаров
- `frontend/product.html` — страница товара
- `frontend/app.js` — логика загрузки каталога
- `frontend/product.js` — логика загрузки отдельного товара
- `frontend/styles.css` — базовые стили

## Что есть на каталоге
- строка поиска
- select по общей категории
- select по бренду
- select по статусу
- кнопка `Обновить`
- сетка карточек товаров
- кнопки пагинации

Карточка товара показывает:
- изображение
- бренд
- название
- категории
- размер
- состояние
- внутренний статус товара
- цену в исходной валюте
- ссылку на страницу товара

## Что есть на странице товара
- название
- бренд
- категории
- пол
- размер
- состояние
- замеры
- цена
- описание
- ссылка на исходный товар
- галерея изображений

## Откуда frontend берет данные
- каталог: `GET /api/products`
- значения фильтров: `GET /api/filter-options`
- страница товара: `GET /api/products/{source_product_id}`

## Ограничения текущей версии
- нет сортировки
- стили базовые, без адаптивной проработки уровня production
