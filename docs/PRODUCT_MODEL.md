# PRODUCT MODEL

## Основные поля товара

- id
- title
- brand
- category
- subcategory
- size
- condition
- price_original
- currency
- price_rub
- delivery_estimate
- description
- source
- product_url
- image_url

## Описание полей

- `id` — уникальный идентификатор товара
- `title` — название товара
- `brand` — бренд
- `category` — основная категория
- `subcategory` — подкатегория
- `size` — размер
- `condition` — состояние товара
- `price_original` — исходная цена
- `currency` — валюта исходной цены
- `price_rub` — цена в рублях
- `delivery_estimate` — примерный срок доставки
- `description` — описание товара
- `source` — источник данных
- `product_url` — ссылка на исходную страницу товара
- `image_url` — основное изображение товара
