# RINKAN FIRST SOURCE

## Решение
Для первой версии проекта первым источником для парсинга выбран **RINKAN**.

## Зачем это решение принято сейчас
- проекту нужен один фиксированный источник для MVP
- это позволяет проектировать парсер, структуру данных и API не абстрактно, а под реальный сайт
- остальные источники будут рассматриваться после первой рабочей версии

## Что это означает для следующего этапа
Далее разработка должна идти от источника RINKAN:
- определить, какие поля реально доступны на карточке товара
- зафиксировать структуру сущности товара
- подготовить первый рабочий парсер
